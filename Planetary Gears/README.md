# Planetary Gear System

## Overview

This project showcases the CAD design and assembly of a planetary gear system developed to study compact power transmission mechanisms used in automotive, aerospace, and robotic applications. The design focuses on accurate gear geometry, proper meshing, and realistic assembly constraints while maintaining manufacturability.

## Objectives

- Design a complete planetary gear assembly
- Model accurate involute gear profiles
- Ensure proper gear meshing and alignment
- Understand speed reduction and torque multiplication principles
- Create a modular assembly suitable for manufacturing and future analysis

## Components

- Sun Gear
- Planet Gears
- Planet Carrier
- Ring Gear
- Assembly
