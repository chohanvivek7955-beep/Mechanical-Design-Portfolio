# Quadruped Robot

## Overview

The Quadruped Robot is a four-legged robotic platform developed to study agile locomotion, balance, and mechanical optimization. The project emphasizes lightweight design, modular construction, and efficient force transmission.

## Objectives

- Design a compact and lightweight quadruped
- Optimize joint mechanisms for smooth motion
- Improve structural rigidity while reducing weight
- Develop a modular design for future upgrades

## My Contributions

- Designed mechanical components and assemblies
- Optimized leg geometry for improved stability
- Created manufacturing-ready CAD models
- Improved component accessibility for maintenance
- Assisted in integrating the mechanical system with control architecture
