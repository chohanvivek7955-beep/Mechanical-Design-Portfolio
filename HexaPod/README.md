# Hexapod Robot

## Overview

The Hexapod Robot is a six-legged autonomous robotic platform designed to traverse uneven terrain while maintaining stability and mobility. This project focuses on mechanical design, structural optimization, and robotic locomotion.

## Objectives

- Design a lightweight and modular chassis
- Develop a robust six-legged locomotion mechanism
- Optimize components for 3D printing and manufacturing
- Enable stable walking and obstacle traversal

## My Contributions

- Designed the complete mechanical assembly using Fusion 360
- Optimized structural components for weight and strength
- Applied Design for Manufacturing (DFM) principles
- Designed leg mechanisms using inverse kinematics considerations
- Conducted multiple design iterations to improve performance
- Coordinated subsystem integration with electronics and software teams

## Software Used
