# TurtleBot

## Overview

This project focuses on improving the mechanical design of a TurtleBot platform used for autonomous navigation and SLAM (Simultaneous Localization and Mapping). The work emphasizes chassis optimization, sensor mounting, and overall mechanical reliability.

## Objectives

- Improve chassis stability
- Optimize sensor placement
- Enhance load distribution
- Support autonomous navigation experiments

## My Contributions

- Redesigned the chassis for balanced weight distribution
- Developed custom mounting solutions for sensors and electronics
- Improved drivetrain alignment for smoother movement
- Optimized component placement to simplify assembly and maintenance

## Software Used

- Fusion 360
- ROS
- Gazebo
